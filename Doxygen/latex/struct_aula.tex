\doxysubsection{Detailed Description}
This struct facilitates the carriage of information of a certain Lecture of a certain Class. 

This struct has 4 attributes, week\+Day, the hour of begging, the ending hour and it\textquotesingle{}s type (T if the Class is theorical, TP if it is practical-\/theoric and PL if it is Practical Laboratory
