\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classStudent}{Student}} class represents a student. 

The \mbox{\hyperlink{classStudent}{Student}} class includes attributes for student number, name, and a vector to keep track of the courses (UC) and classes the student is enrolled in.
